클래스 기초 개념 정리\n
자동차, 사람, 회사 등 독립되고 구분되는 속성과 기능을 가진 프로그램의 단위를 객체(object)라고 한다.
이러한 객체들을 생성하기 위한 틀, 설계도와 같은 역할을 하는것이 클래스(class)이다.
다만 객체의 모든것을 클래스에 옮기진 않고 추상화라는 프로그램에 필요한 특성만을 구현하는 과정을 통해 만들어지게 된다.
클래스는 기본적으로 2가지의 구성요소가 있는데 필드와 메소드이다.
필드는 클래스내에서 정의된 변수를 뜻한다. 예를 들어 학생이라는 클래스가 있다면 
이 클래스의 필드로는 학생번호, 이름, 성적, 수강과목 등을 만들 수 있다. 필요한 것을 만들면 된다.
두번째로 메소드는 특정 작업을 수행하도록 이름을 붙여둔 코드의 덩어리이다. 수학에서의 함수라고 생각하면 편리하다.
예를들어 f(x) = 2x라는 함수가 있다면 이 함수의 작업은 입력값을 2배로 출력해주는 역할이다.
이런 간단한 작업부터 복잡한 작업까지 미리 만들어두고 필요할 때 만들어서 쓰는 용도이다.
메소드중에는 생성자라는 특수한 메소드가 존재하는데 객체를 생성할때 딱 한번만 호출되는 특수한 메소드이다.
기본적으로 생성자를 생성하지 않으면 기본 메소드가 프로그램 내에 생성되어있고, 원하는 대로 생성자를 만들 수도 있다.
